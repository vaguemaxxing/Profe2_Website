Level 2 DFD - Sales Process

Level 2 DFD - Inventory Process

Process Flow Description

  1. 2.1 Receive Order from Sales: Kitchen staff retrieves order slips from queue
  2. 2.2 Prepare Food: Cook meals according to order specifications
  3. 2.3 Track Ingredient Usage: Record consumption of ingredients during food preparation
  4. 2.4 Check Stock Levels: Monitor inventory and alert manager when supplies run low
  5. 2.5 Create Purchase Order: Manager approves and creates orders for suppliers
  6. 2.6 Process Supplier Payment: Handle payment transactions for purchased supplies using funds from Finance
  7. 2.7 Receive & Update Inventory: Accept deliveries and update inventory records
  8. 2.8 Process Employee Payroll: Manager approves payroll, funds come from Finance, paychecks go to employees
